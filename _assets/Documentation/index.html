Installing WordPress

WordPress is well-known for its ease of installation. Under most circumstances installing WordPress is a very simple process and takes less than five minutes to complete. Many web hosts now offer tools (e.g. Bluehost, mediatemple) to automatically install WordPress for you. However, if you wish to install WordPress yourself, click below button to read about Famous 5 Minute Installation.

5 Minute Installation Guide

Getting Started

To install this theme you must have a working version of WordPress already installed.
For information in regard to installing the WordPress platform, please see the WordPress Codex – http:// codex.wordpress.org/Installing_WordPress

Installing Theme

When you are ready to install a theme, you must first upload the theme files and then activate the theme itself.

The theme files can be uploaded in two ways:

FTP Upload: Using your FTP program, upload the non-zipped theme folder into the /wp-content/themes/ folder on your server.

WordPress Upload: Navigate to Appearance > Add New Themes > Upload. Go to browse, and select the zipped theme folder. Hit “Install Now” and the theme will be uploaded and installed.

Once the theme is uploaded, you need to activate it.

Go to Appearance > Themes and activate your chosen theme.

Plugin Activation

This theme required some plugins. You need to install:

  • MD PAGE BUILDER: Enable Drag&Drop Page Builder + Shortcodes.
  • MD CUSTOM POST: Custom Post Types.
  • REVOLUTION SLIDER: Premium Slider with backend interface.
  • CONTACT FORM 7: Easy manage your forms.
  • OAUTH TWITTER FEED FOR DEVELOPERS: Twitter API.

After that, YOU ARE READY TO USE ZETA!

Setting the Menu

Zeta uses WordPress 3.0 Custom Menus

  1. Go to Appearance > Menus.
  2. Create a new menu.
  3. Add your created pages to it and save it.
  4. Use the drop down box to select the Menu and the Location.
  5. Save and your menu is ready.
You can enable/disable footer menu in the Theme Panel.

For add icons to menu links, simply add <i class="icon-NAME"></i> in the label section.
You can find a list of available icons on the demo site.
Remember to enable MENU ICONS options in the Theme Panel.

Setting the Footer

To add widgets to theme's footer follow these steps:

  1. Go to Appearance > Widgets.
  2. Open up the footer area position on the right area.
  3. Drop widgets in the appropriate position.
You can choose the number of areas in the Theme Panel.

MD Theme Options

The Theme Options the is home to all of the theme's extensive configuration options. To view the panel, navigate to the Theme Panel icon in the left-hand menu. Here you have the ability to alter many core aspects that make up how your theme looks and behaves.

Each option has a description that explain the behavior.

MD Page Options

The Page Options panel is present in each page of your site. After create a page, you'll see the panel at the bottom of the page.
From here you can easily customize the header of your page and add a slider.

For custom post types, like Portfolio and Team you can set from here the details of the item

For blog posts, you can add media from here based on your post format.

Each option has a description that explain the behavior.

MD Page Builder

The Page Builder allows you to create stunning template without coding.

When you are in a page, simply click on add section/element and select the element that you want to put in your page.
A step-by-step form helps you to customize your element.
You can save your template and use it again in another page.

Each option has a description that explain the behavior.

MD Shortcodes

Shortcodes are one of the most popular element of WordPress.

To add a shortcode to your page, click on the icon in the TinyMce Editor and follow the steps.

Each option has a description that explain the behavior.
